# 📄 Currículo Fictício em HTML e CSS

Este projeto consiste em uma página web de currículo desenvolvida utilizando **HTML** e **CSS**, com o objetivo de demonstrar habilidades básicas em desenvolvimento frontend, estruturação de páginas e estilização.

> ⚠️ Este é um currículo fictício, criado apenas para fins de estudo e demonstração de habilidades.

---

## 🚀 Funcionalidades

- 📄 Estruturação completa de um currículo em HTML
- 🎨 Estilização com CSS externo
- 🔗 Links interativos (email e redes sociais)
- 🖼️ Inserção de imagem de perfil
- 📱 Organização de conteúdo em seções:
  - Contato
  - Objetivo profissional
  - Formação académica
  - Experiência profissional
  - Habilidades

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de:

- Praticar a criação de páginas web estáticas
- Aprender a estruturar conteúdos com HTML
- Aplicar estilos utilizando CSS
- Desenvolver organização visual de informações
